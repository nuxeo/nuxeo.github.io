nuxeo/5.7.1 Release Notes

New features in DAM

DAM is customizable with Nuxeo Studio

Redesign from the ground up using ContentView, Layouts and widgets. It makes every part of DAM screen (Search panel, result panel, asset view) easy to customize using Nuxeo Studio.

Back Navigation

We've enhance the back navigation of your browser using HTML5 pushState.

Permlink available for any Asset

Every Asset now have an accessible and sharable permlink for you to share.

Use tags in searches

You can now add tags in your search form. Whether you're using the advanced search form, any faceted search, a content view query or a query operation. It works because we've added tag suppport to NXQL. screenshot?
