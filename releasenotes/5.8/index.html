Nuxeo Platform 5.8 Release Notes

Nuxeo Platform 5.8 is now available for download. This is the first LTS version, using the newly launched versioning system, with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release:

  • For Users
    • Nuxeo Drive
    • DAM
    • Workflow
    • Mobile
    • CSV Importer
    • User Experience
    • Search
    • Quota
    • BIRT
    • Jenkins Report Module
    • Video
    • Translations
  • For Administrators
    • Monitoring
    • Multi-Tabs Navigation
    • Right Audit
    • Kerberos Login Module
    • MarketPlace/Update Center
    • Blogs and Websites Packaging
    • Database Support
    • Nuxeo.conf New Parameters
  • For Developers
    • Content Automation
    • Document Oriented REST API
    • CMIS
    • New operations
    • CSS 3 Support in Theme Engine
    • UI Style Guide
    • UI Framework
    • Content Views
    • CORS support
    • Scan Importer
    • Login Page
    • Long Running Listener
    • Move To Tomcat 7
    • Nuxeo Visible Content Store (VCS)
    • Redis
    • PDF Signing
    • Mockito Integration
    • Localization
    • Miscellaneous
  • Upgrade Notes
  • Known Issues
