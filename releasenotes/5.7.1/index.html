Nuxeo Platform Release Notes

Nuxeo Platform 5.7.1 is now available for download (1). This is the first Fast Track version, using the newly launched versioning system (2), with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release:

    • DAM
    • Mobile
    • Importer CSV
    • User Experience
    • Search
    • Quota
    • Birt
    • Content Routing
    • Jenkins Report Module
  • For Administrators
  • For Developers
  • Upgrade Notes
  • Known Issues
