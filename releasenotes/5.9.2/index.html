Nuxeo Platform 5.9.2 Release Notes

Nuxeo Platform 5.9.2 is now available for download. This is the second Fast Track for the next LTS release, using the versioning system launched last year, with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release:

  • For Users
    • DAM Enhancement
    • Nuxeo Drive
    • Suggestion Widgets
  • For Administrators
    • Private Marketplace Channel
    • New Parameters for Nuxeo.conf
  • For Developers
    • JavaScript SDK
    • Migration to Maven 3
    • Token Based Authentication Management & OAuth 2.0 Support
    • JSF State and HTTP Session Debug
    • Java 8 Support
    • NXQL QUery and Fetch
  • Upgrade Notes
  • Known Issues
