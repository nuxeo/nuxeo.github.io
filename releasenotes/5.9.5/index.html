Nuxeo Platform 5.9.5 Release Notes

Nuxeo Platform 5.9.5 is now available for download. This is the fifth Fast Track for the next LTS release, using the versioning system launched last year, with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release:

  • For Users
    • Insert a Video in a Note
    • Photoshop Mime-types Correctly Detected
  • For Administrators
    • Logging Slow Queries
    • Java version is checked at server startup
    • Elasticsearch
  • For Developers
    • MongoDB Backend
    • REST API improvements
    • Elasticsearch
    • Business Days Management
    • Generic Video Class Converter
    • SCIM 1.1 Implementation
    • Okta SSO
    • Duo Web Two Factors authentication plugin
  • Upgrade Notes
